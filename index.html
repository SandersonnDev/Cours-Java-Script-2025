Cours de JavaScript pour Débutants (2025)

Chapitre 1 : Introduction à JavaScript

1.1 Qu’est-ce que JavaScript ?

  • JavaScript est un langage de programmation qui permet de rendre les pages web interactives.
  • Il est exécuté directement dans le navigateur (comme Chrome, Firefox, Edge).
  • Il est utilisé pour :
    • Ajouter des animations.
    • Réagir aux actions de l’utilisateur (clics, saisies, etc.).
    • Manipuler le contenu HTML et CSS.

1.2 Où écrire du JavaScript ?

Théorie :

  • JavaScript peut être écrit directement dans un fichier HTML ou dans un fichier séparé avec l’extension .js.
  • Pour inclure JavaScript dans une page HTML, on utilise la balise <script>.

Exemple :

Voici un exemple simple où JavaScript est inclus directement dans une page HTML.

<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Premier Script JS</title>
</head>
<body>
    <h1>Bienvenue dans le monde de JavaScript !</h1>

    <!-- Balise script : permet d'écrire du code JavaScript -->
    <script>
        // Affiche un message dans la console du navigateur
        console.log("Bonjour, JavaScript !");
    </script>
</body>
</html>

Exercice pratique :

  1. Ouvrez votre éditeur de texte (comme Visual Studio Code).
  2. Créez un fichier HTML.
  3. Écrivez une structure HTML de base (comme dans l’exemple ci-dessus).
  4. Ajoutez une balise <script> dans le <body>.
  5. Écrivez un code JavaScript pour afficher un message dans la console avec console.log().

1.3 Inclure un fichier JavaScript externe

Théorie :

  • Pour mieux organiser le code, il est recommandé d’écrire JavaScript dans un fichier séparé.
  • On inclut ce fichier dans le HTML avec l’attribut src de la balise <script>.

Exemple :

  1. Créez un fichier nommé script.js et écrivez le code suivant :
//Affiche un message dans la console
console.log("Bonjour depuis un fichier externe !");
  1. Dans votre fichier HTML, incluez ce fichier avec la balise <script> :
<script  src="script.js"></script>

Exercice pratique :

  1. Créez un fichier JavaScript nommé script.js.
  2. Écrivez un code JavaScript pour afficher un message dans la console.
  3. Dans votre fichier HTML, incluez ce fichier avec la balise script.

Chapitre 2 : Les bases de JavaScript

2.1 Les variables

Théorie :

  • Une variable est un conteneur qui stocke une valeur.
  • En JavaScript, on utilise :
    • let : Pour déclarer une variable modifiable.
    • const : Pour déclarer une variable constante (non modifiable).
    • var : Ancienne méthode (à éviter dans les nouveaux projets).

Exemple :

// Déclaration d'une variable modifiable
let nom = "Alice"; // Chaîne de caractères (String)
let age = 25;      // Nombre (Number)
let estEtudiant = true; // Booléen (Boolean)

// Affichage des variables dans la console
console.log("Nom :", nom);
console.log("Âge :", age);
console.log("Est étudiant :", estEtudiant);

Exercice pratique :

  1. Créez un fichier JavaScript.
  2. Déclarez trois variables :
    • Une variable pour votre nom (type String).
    • Une variable pour votre âge (type Number).
    • Une variable pour indiquer si vous êtes débutant en JavaScript (type Boolean).
  3. Affichez ces variables dans la console avec console.log().

2.2 Les types de données

Théorie :

  • Les principaux types de données en JavaScript sont :
    • String : Chaîne de caractères (ex. "Bonjour")
    • Number : Nombre (ex. 42)
    • Boolean : Vrai ou faux (true ou false)
    • Null : Valeur vide
    • Undefined : Variable non définie

Exemple :

let texte = "Bonjour"; // String
let nombre = 42;       // Number
let estVrai = true;    // Boolean
let vide = null;       // Null
let nonDefini;         // Undefined (pas de valeur attribuée)

console.log(texte, nombre, estVrai, vide, nonDefini);

Exercice pratique :

  1. Créez une variable pour chaque type de donnée mentionné ci-dessus.
  2. Affichez-les dans la console avec console.log().

2.3 Les opérations de base

Théorie :

  • Les opérateurs permettent de manipuler les données :
    • Addition : +
    • Soustraction : -
    • Multiplication : *
    • Division : /

Exemple :

let a = 10;
let b = 5;

console.log("Addition :", a + b);       // 15
console.log("Soustraction :", a - b);  // 5
console.log("Multiplication :", a * b); // 50
console.log("Division :", a / b);      // 2

Exercice pratique :

  1. Déclarez deux variables x et y avec des valeurs numériques.
  2. Calculez leur somme, différence, produit, et quotient.
  3. Affichez les résultats dans la console.

Chapitre 3 : Les conditions

3.1 Les structures conditionnelles

Théorie :

  • Les conditions permettent d’exécuter du code en fonction d’une situation.
  • La structure de base est la suivante :
if (condition) {
    // Code à exécuter si la condition est vraie
} else {
    // Code à exécuter si la condition est fausse
}

Exemple :

let age = 18;

if (age >= 18) {
    console.log("Vous êtes majeur."); // S'exécute si la condition est vraie
} else {
    console.log("Vous êtes mineur."); // S'exécute si la condition est fausse
}

Explications :

  1. if (age >= 18) : Vérifie si la variable age est supérieure ou égale à 18.
  2. Si la condition est vraie, le code dans les accolades {} après if est exécuté.
  3. Sinon, le code dans les accolades après else est exécuté.

3.2 Les conditions multiples

Théorie :

  • Pour tester plusieurs conditions, on peut utiliser else if :
if (condition1) {
    // Code si condition1 est vraie
} else if (condition2) {
    // Code si condition2 est vraie
} else {
    // Code si aucune condition n'est vraie
}

Exemple :

let heure = 14;

if (heure < 12) {
    console.log("Bonjour !");
} else if (heure < 18) {
    console.log("Bon après-midi !");
} else {
    console.log("Bonsoir !");
}

Exercice pratique :

  1. Déclarez une variable note (entre 0 et 20).
  2. Écrivez un programme qui affiche :
    • “Très bien” si la note est supérieure ou égale à 16.
    • “Bien” si la note est entre 12 et 15 inclus.
    • “Passable” si la note est entre 10 et 11 inclus.
    • “Échec” si la note est inférieure à 10.

Chapitre 4 : Les boucles

4.1 La boucle for

Théorie :

  • Une boucle permet de répéter une action plusieurs fois.
  • La boucle for a la structure suivante :
for (initialisation; condition; incrément) {
    // Code à répéter
}
  • Initialisation : Déclare une variable pour démarrer la boucle.
  • Condition : Vérifie si la boucle doit continuer.
  • Incrément : Modifie la variable après chaque itération.

Exemple :

for (let i = 1; i <= 5; i++) {
    console.log("Itération :", i); // Affiche les nombres de 1 à 5
}

Explications :

  1. let i = 1 : Initialise la variable i à 1.
  2. i <= 5 : Continue la boucle tant que i est inférieur ou égal à 5.
  3. i++ : Incrémente i de 1 après chaque itération.

4.2 La boucle while

Théorie :

  • La boucle while répète une action tant qu’une condition est vraie.
while (condition) {
    // Code à répéter
}

Exemple :

let compteur = 1;

while (compteur <= 5) {
    console.log("Compteur :", compteur); // Affiche les nombres de 1 à 5
    compteur++; // Incrémente la variable
}

Exercice pratique :

  1. Écrivez une boucle for qui affiche les nombres de 1 à 10.
  2. Écrivez une boucle while qui affiche uniquement les nombres pairs entre 1 et 10.

Chapitre 5 : Les fonctions

5.1 Définir une fonction

Théorie :

  • Une fonction est un bloc de code réutilisable.
  • Elle peut prendre des paramètres et retourner un résultat.
function nomDeLaFonction(param1, param2) {
    // Code
    return resultat;
}

Exemple :

function addition(a, b) {
    return a + b; // Retourne la somme de a et b
}

let resultat = addition(3, 7); // Appelle la fonction avec 3 et 7
console.log("Résultat :", resultat); // Affiche 10

Explications :

  1. function addition(a, b) : Définit une fonction nommée addition avec deux paramètres a et b.
  2. return a + b : Retourne la somme des deux paramètres.
  3. addition(3, 7) : Appelle la fonction avec les valeurs 3 et 7.

5.2 Les fonctions anonymes et fléchées

Théorie :

  • Une fonction anonyme est une fonction sans nom.
  • Une fonction fléchée est une version simplifiée des fonctions anonymes.

Exemple :

// Fonction anonyme
let multiplier = function (x, y) {
    return x * y;
};

// Fonction fléchée
let multiplierFleche = (x, y) => x * y;

console.log(multiplier(3, 4)); // Affiche 12
console.log(multiplierFleche(3, 4)); // Affiche 12

Exercice pratique :

  1. Créez une fonction nommée saluer qui prend un paramètre nom et affiche “Bonjour, [nom] !”.
  2. Créez une fonction fléchée qui calcule le carré d’un nombre.

Chapitre 6 : Manipulation du DOM

6.1 Sélectionner et modifier des éléments

Théorie :

  • Le DOM (Document Object Model) représente la structure HTML sous forme d’objet.
  • Avec JavaScript, vous pouvez :
    • Sélectionner des éléments HTML.
    • Modifier leur contenu ou leur style.

Exemple :

<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Manipulation du DOM</title>
</head>
<body>
    <h1 id="titre">Titre original</h1>
    <button id="bouton">Changer le titre</button>

    <script>
        // Sélectionne les éléments HTML
        let titre = document.getElementById("titre");
        let bouton = document.getElementById("bouton");

        // Ajoute un événement au bouton
        bouton.addEventListener("click", function () {
            titre.textContent = "Titre modifié par JavaScript !"; // Change le texte du titre
        });
    </script>
</body>
</html>

Exercice pratique :

  1. Créez une page HTML avec un titre <h1> et un bouton <button>.
  2. Ajoutez un script pour changer le texte du titre lorsque vous cliquez sur le bouton.

Chapitre 7 : Les tableaux (Arrays)

7.1 Qu’est-ce qu’un tableau ?

Théorie :

  • Un tableau est une structure de données qui permet de stocker plusieurs valeurs dans une seule variable.
  • Les valeurs dans un tableau sont appelées éléments et sont accessibles via leur index (commençant à 0).

Exemple :

// Déclaration d'un tableau
let fruits = ["Pomme", "Banane", "Orange"];

// Accéder aux éléments du tableau
console.log(fruits[0]); // Affiche "Pomme"
console.log(fruits[1]); // Affiche "Banane"

// Ajouter un élément au tableau
fruits.push("Mangue");
console.log(fruits); // ["Pomme", "Banane", "Orange", "Mangue"]

// Supprimer le dernier élément
fruits.pop();
console.log(fruits); // ["Pomme", "Banane", "Orange"]

Exercice pratique :

  1. Créez un tableau nommé animaux contenant trois noms d’animaux.
  2. Ajoutez un nouvel animal à la fin du tableau.
  3. Supprimez le premier élément du tableau.
  4. Affichez le tableau final dans la console.

Chapitre 8 : Les objets

8.1 Qu’est-ce qu’un objet ?

Théorie :

  • Un objet est une collection de propriétés, où chaque propriété est une paire clé-valeur.
  • Les objets permettent de regrouper des données et des fonctions liées.

Exemple :

// Déclaration d'un objet
let personne = {
    nom: "Alice",
    age: 25,
    estEtudiant: true,
    saluer: function () {
        console.log("Bonjour, je m'appelle " + this.nom);
    }
};

// Accéder aux propriétés
console.log(personne.nom); // Affiche "Alice"
console.log(personne.age); // Affiche 25

// Appeler une méthode
personne.saluer(); // Affiche "Bonjour, je m'appelle Alice"

Exercice pratique :

  1. Créez un objet nommé voiture avec les propriétés suivantes :
    • marque : une chaîne de caractères.
    • modele : une chaîne de caractères.
    • annee : un nombre.
    • demarrer : une méthode qui affiche “La voiture démarre”.
  2. Affichez les propriétés de l’objet dans la console.
  3. Appelez la méthode demarrer.

Chapitre 9 : Les événements

9.1 Qu’est-ce qu’un événement ?

Théorie :

  • Un événement est une interaction de l’utilisateur avec la page (clic, saisie, survol, etc.).
  • On peut écouter ces événements avec la méthode addEventListener.

Exemple :

<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Événements</title>
</head>
<body>
    <button id="monBouton">Cliquez-moi</button>

    <script>
        // Sélectionner le bouton
        let bouton = document.getElementById("monBouton");

        // Ajouter un événement de clic
        bouton.addEventListener("click", function () {
            console.log("Bouton cliqué !");
        });
    </script>
</body>
</html>

Exercice pratique :

  1. Créez une page HTML avec un bouton <button> et un champ de texte <input>.
  2. Ajoutez un événement click au bouton pour afficher un message dans la console.
  3. Ajoutez un événement input au champ de texte pour afficher la valeur saisie dans la console.

Chapitre 10 : Exercice global

Objectif :

Créer une petite application interactive qui combine tout ce que nous avons appris.

Instructions :

  1. Créez une page HTML avec les éléments suivants :

    • Un titre <h1> qui affiche “Gestionnaire de tâches”.
    • Un champ de texte <input> pour saisir une tâche.
    • Un bouton <button> pour ajouter la tâche.
    • Une liste <ul> pour afficher les tâches.

  2. Ajoutez un script JavaScript pour :

    • Ajouter une tâche à la liste lorsque l’utilisateur clique sur le bouton.
    • Supprimer une tâche de la liste lorsque l’utilisateur clique dessus.

Code attendu :

Voici un exemple de ce que vous pourriez écrire :

<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Gestionnaire de tâches</title>
</head>
<body>
    <h1>Gestionnaire de tâches</h1>
    <input type="text" id="tacheInput" placeholder="Entrez une tâche">
    <button id="ajouterBouton">Ajouter</button>
    <ul id="listeTaches"></ul>

    <script>
        // Sélectionner les éléments HTML
        let input = document.getElementById("tacheInput");
        let bouton = document.getElementById("ajouterBouton");
        let liste = document.getElementById("listeTaches");

        // Ajouter une tâche
        bouton.addEventListener("click", function () {
            let tache = input.value; // Récupérer la valeur du champ
            if (tache !== "") {
                let li = document.createElement("li"); // Créer un nouvel élément <li>
                li.textContent = tache; // Ajouter le texte de la tâche
                liste.appendChild(li); // Ajouter l'élément à la liste

                // Ajouter un événement pour supprimer la tâche
                li.addEventListener("click", function () {
                    liste.removeChild(li); // Supprimer la tâche
                });

                input.value = ""; // Réinitialiser le champ de texte
            }
        });
    </script>
</body>
</html>

Ce que vous avez révisé :

  1. HTML et DOM : Création et manipulation d’éléments HTML.
  2. Variables : Stocker et utiliser des données.
  3. Conditions : Vérifier si une tâche est vide avant de l’ajouter.
  4. Boucles : (Implicite dans la gestion des tâches via les événements).
  5. Fonctions : Utilisation de fonctions anonymes pour les événements.
  6. Tableaux et Objets : (Peuvent être ajoutés pour gérer les tâches de manière plus avancée).
